Одностраничное веб-приложение для студентов и сотрудников университета:

- `index.html` — точка входа, интерфейс с расписанием, новостями, клубами и сервисами.  
- `style.css` — стили и адаптивная верстка под мобильные устройства.  
- `script.js`, `student_inter.js`, `staff.js` и др. — логика интерфейса студента/сотрудника, планировщик задач, работа с данными.  
- `supabase-client.js` — клиент для Supabase (подключение через JS SDK и REST-запросы к таблицам `news`, `clubs`, `events`, `schedule` и т.д.).  
- `auth.js`, `data.js` — авторизация, мок-данные университетов, пользователей, расписания и т.п.

> **Важно:** проект — полностью фронтендовый, без Python-бекенда.  
> Python нужен только для простого HTTP-сервера локально и внутри Docker-контейнера.

---

## 1. Локальный запуск без Docker

### 1.1. Вариант 1 — просто открыть в браузере

Самый простой способ:

1. Склонировать/распаковать проект в любую папку.
2. Дважды кликнуть по `index.html` → откроется в браузере.

**Минусы:**  
некоторые браузеры могут ограничивать работу с `localStorage`/`fetch` при открытии через `file://`. Рекомендуется вариант с локальным HTTP-сервером (ниже).

---
